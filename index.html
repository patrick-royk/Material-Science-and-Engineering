Programming Cell Behaviors in Python

Your script evaluates inside every individual grid box coordinate during system step advances. It expects a standard functional output return value sequence.

Available Variables:
  • self: The current binary status state of the evaluated target block (1 or 0).
  • neighbors: A standard Python list containing the active binary states of all adjacent neighbors (e.g., [0, 1, 1, 0...]).
Examples: 
# High Stability Rules (Conway Variant)
count = sum(neighbors)
if count == 3: return 1
if count == 2: return self
return 0
